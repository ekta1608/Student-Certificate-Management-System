Certificate Management System

Description
A web-based certificate management system designed to store, manage, and retrieve student certificates digitally. The system supports role-based access for students and administrators, enabling efficient handling of academic and achievement certificates.

Features
- Student and admin authentication
- Secure storage of certificate details
- Upload and management of certificate links
- Role-based access control
- REST APIs for certificate operations

Technologies Used
- JavaScript
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT (authentication)
- HTML/CSS (basic frontend)
